# 管理4C模型

------

## 介绍

管理的4C模型是提升团队执行力的有效方法，

它包含四个行动：

1. 布置任务要清晰(Clarity)：确保员工100%理解工作
2. 承诺工作要真心(Commitment)：发自内心的承诺
   胜任工作要到位(competence)：不仅是能力，还
   有态度
3. 控制工作要全面(Control)：事前、事中、事后的控
   制

## 何时用

作为团队的领导，要提升团队执行力时，可以采取
4C模型。
