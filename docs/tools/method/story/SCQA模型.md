# SCQA模型

------

## 介绍

SCQA模型是一个结构化表达工具，是麦肯锡顾问芭芭拉•明拓在《金字塔原理》中提出，

由四个部分组成：

S (Situation）情景——由大家都熟悉的情景、事实引入

C (Complication)冲突—实际情况往往和我们的要求有冲突

Q (Question) 疑问—怎么办？

A (Answer)回答一我们的解决方案是

## 何时用

当你被要求临场发言，需要快速组织语言内容时，可套用这个模型。
