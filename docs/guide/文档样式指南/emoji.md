# Emoji 表情

美政的自定义组件。

在文档中添加 Emoji 表情，增加文档的可读性和情感体验。

:::info 提示

1、如果 Emoji 表情为内容开头，建议在 Emoji 表情后面增加一个空格，以增加可读性。<br />2、如果 Emoji 表情为内容中间，建议在 Emoji 表情前面和后面都增加一个空格，以增加可读性。

:::
